# hse_stat


## Getting Started

### Download links:

SSH clone URL: git@github.com:OLavrik/hse_stat.git

HTTPS clone URL: https://github.com/OLavrik/hse_stat.git


### Installation

- conda create -n hse python=3.8 
- conda activate hse
- pip install -r ./requirements.txt



### Run

- conda activate hse
- python main.py  -i ./in.txt -o ./out.txt
